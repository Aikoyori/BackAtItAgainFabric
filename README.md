# Back At It Again! for Fabric

Fabric mod to re-add old and removed features from Minecraft back into the game.

# Features
- Nether Reactor Core along with the Nether Spire
- Glowing Obsidian
- Legacy Stonecutter
- info_update and info_update2 blocks
